------ Problem ------

در Vue 2  و هنگام با آن vue-router 3 زمانی که روت تغییر می دهیم همچنان روت home یا / دارای کلاس active می ماند. برای حل این مشکل 3 راه حل ارائه شده است.



1.    به router-link مربوط به / یک اتریبوت اضافه کنیم با نام exact

2.    کلاس ها   router-link-exact-active  و active را با ویژگی active-class و exact-active-class خود ویو تغییر دهیم.

3.    استایل های خود را برای لینک active براساس router-link-exact-active تعریف کنیم.

 

 

// Method 1

<router-link to="/"   exact >Home</router-link>

 

// Method 2

<router-link    to="/"   active-class="router-link-exact-active"  exact-active-class="active"  >Home</router-link>

 

// Method 3

.router-link-exact-active {

  background:#ccc;

}
