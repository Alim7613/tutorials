برای استفاده بهتر از ویو می توانید افزونه  Vue Devtools   را به مرورگر خود اضافه کنید. لازم به ذکر است که این افزونه تنها در حالت  Development به درستی کار می کند و زمانی که پروژه تکمیل شد و در حالت  Production قرار گرفت دیگر کار نخواهد کرد. 

 

زمانی که برنامه را اجرا می کنید در کنسول دوتا اعلان می آید اولی می گوید که شما در حال استفاده از نسخه محصوص توسعه هستید و اگر می خواهید سایت را اپلود کنید برای سرعت بیشتر در vue3از  (*.prod.js) استفاده کنید.

 

Make sure to use the production build (*.prod.js) when deploying for production.

دومی هم می گوید که افزونه را نصب کنید برای مدیریت بهتر پروژه ها.



کدهایی که در فایل HTML خود قرار می دهیم (در اصل آن قسمتی که با el به یک نمونه Vue متصل شده باشند) توسط Vue دریافت شده و در پشت پرده پردازش می شود. این پردازش بر اساس template engine فریم ورک Vue انجام می شود که فعلا با آن کاری نداریم. نکته مهم این است که Vue کدهای HTML شما را گرفته و درون حافظه خود آن ها را پردازش می کند. سپس یک فایل نهایی HTML می سازد و آن کدهای نهایی را به کاربر می دهد.



دقت کنید که مقدار وارد شده بین  {{ }}  باید مقداری باشد که توانایی تبدیل شدن به رشته (string) را دارا باشد.

داخل تگ ها html نمی توانیم از {{}}  استفاده کنیم و فقط بیرون تگ ها .        برای داخل تگ ها باید بایند ( : ) کنیم.   

 

نکته مهم : شما می توانید از این روش برای bind کردن تمام attribute های HTML استفاده کنید

(class - style - data - href - src - alt -….)
